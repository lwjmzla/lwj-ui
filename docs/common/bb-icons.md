# Icon 图标

基于 `element-ui` 使用的 `icon` 集合。

## 使用方法

```html
<!-- 使用 el-icon 标签来包裹 SVG 图标 -->
<template>
  <div>
    <el-icon :size="size" :color="color">
      <edit />
    </el-icon>
    <!-- 也可以直接使用图标标签，无需父标签包裹 -->
    <edit />
  </div>
</template>
```
